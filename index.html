  • Sobre el proyecto
  • Disclaimer

Este proyecto colaborativo arrancó con la idea de tener un lugar mapeado donde encontrar lugares donde haya recreativas/arcades/fichines. Podés colaborar desde el repo github disponible en Mapa De Fichines o en el instagram mapafichines

El código es un fork del proyecto original de github.com/vlasvlasvlas/mapadefichines que se basa proyecto Bootleaf, de licencia MIT, y disponible en GitHub Bootleaf.

"Los datos proporcionados en este sitio son únicamente para fines informativos y de planificación. No se implica ni se pretende ninguna garantía de exactitud o completitud. Toda la información en este mapa está sujeta a variaciones y correcciones.
